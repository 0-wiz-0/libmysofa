libmysofa

C library to read HRTFs if they are stored in the AES69-2015 format.

Welcome to the Libmysofa Page

This is a set of C functions to read SOFA files, if they contain head related transfer functions (HRTF) stored according to the AES69-2015 standard. It is a light weight library having a minimal memory foodprint.

Instead of the libhdf and libnetcdf, we are implementing a simple HDF5 reader following the HDF5 specification. For example, a SOFA to JSON converter written with NETCDF C++ has a size of 798387 bytes on Linux 64bit; written with libmysofa, it has a size of 90625.

Usage

Libmysofa is for C programs. Add a #include <mysofa.h>. You read a SOFA file with one function call (mysofa_open and mysofa_close). To get HRTF filters for given positions with an efficient lookup and interpolation algorithm, call mysofa_getfilter (see README.md).
